# ChurnIQ

An end-to-end customer churn prediction project built with Python, Machine Learning, Streamlit, and LLM-powered customer retention recommendations.

## Features

- Exploratory Data Analysis (EDA)
- Feature Engineering
- Customer Churn Prediction
- Model Evaluation
- Interactive Streamlit Dashboard
- AI-generated retention recommendations

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Streamlit
- OpenAI API
- Git & GitHub
